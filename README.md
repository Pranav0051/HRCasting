
  # Luxury Jewellery Website UI

  This is a code bundle for Luxury Jewellery Website UI. The original project is available at https://www.figma.com/design/muIRhds5rvUyqBIQ4gB1pN/Luxury-Jewellery-Website-UI.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the frontend development server.

  Run `npm run server` to start the backend Node.js server on port 4000.

  For local development with live backend restarts, use `npm run dev:server`.
